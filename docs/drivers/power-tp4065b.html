drv_power_tp4065b

ADC-based battery monitor for LiPo cells charged by the TP4065B. Uses ESP-IDF oneshot ADC API.

Overview Proven

HAL TypePower
ChipTP4065B
InterfaceADC
DescriptionReads raw ADC voltage and maps it to a percentage using a 7-point piecewise linear LiPo discharge curve.

ThistleOS

An open-source operating system for ESP32-S3 devices. Dynamic app loading, swappable hardware drivers, LVGL UI, and LoRa mesh networking — all on a microcontroller.

✎ Edit on GitHub
Status
ThistleOS v0.1.0 is in active development. The kernel, HAL, and driver framework are stable. App signing and the online app store are still being finalized.

What is ThistleOS?

ThistleOS is a micro operating system for the ESP32-S3, initially targeting the LilyGo T-Deck Pro. It brings concepts from mainstream operating systems — loadable apps, a hardware abstraction layer, app permissions, and an app store — down to a device with 512 KB of internal SRAM and 8 MB of PSRAM.

Key design decisions:

  • Drivers are fully swappable. Swapping a display driver means changing one line in a board definition, not touching the kernel.
  • Apps are ELF files on an SD card. The kernel's ELF loader resolves symbols against a syscall table at runtime. No recompilation of the OS needed to install an app.
  • BSD 3-Clause throughout. No GPL components in the ThistleOS core; swap to any commercial use case without license headaches.
  • Recovery OS is Rust. A separate, minimal firmware in ota_0 that can reflash the main OS without a programmer.

Quick Start

Prerequisites: ESP-IDF v5.3+ installed and on your PATH.

# Clone the repo
git clone https://github.com/wan0net/esp32-os
cd esp32-os

# Build for the default board (T-Deck Pro)
idf.py build

# Flash and open serial monitor
idf.py -p /dev/ttyACM0 flash monitor

The SD card layout is documented in sdcard_layout/. Copy the directory contents to a FAT32 SD card and insert it before first boot.

Repository layout

PathPurpose
components/thistle_hal/HAL interface headers — pure vtable structs, no implementation
components/drv_*/One component per hardware driver (display, radio, GPS, etc.)
components/board_*/Board definitions — wire drivers to HAL, define pin numbers
components/kernel/App manager, driver manager, ELF loader, IPC, permissions, OTA
components/ui/LVGL window manager, status bar, toast notifications, themes
components/apps_builtin/Built-in apps: Launcher, Messenger, Navigator, Notes, Reader, Settings
components/shim/Compatibility shims for MeshCore and Arduino-style firmware
app_sdk/SDK for third-party app developers — headers + CMake helpers
recovery/Recovery OS — Rust + esp-rs, lives in ota_0
sdcard_layout/Template for the SD card — apps/, drivers/, themes/, config/

Hardware: LilyGo T-Deck Pro

The primary supported board is the LilyGo T-Deck Pro, an ESP32-S3 device with:

  • 3.1" e-paper display (GDEQ031T10, 300×168, partial refresh)
  • Physical QWERTY keyboard (TCA8418 key matrix)
  • Capacitive touch (CST328)
  • SX1262 LoRa radio (915 MHz / 868 MHz)
  • u-blox MIA-M10Q GPS
  • PCM5102A audio DAC
  • BHI260AP IMU
  • 4G LTE modem (A7682E, optional)
  • microSD card slot

See Board Support for how to port ThistleOS to other ESP32-S3 boards.
